Three problems pop out immediately:a) the test.py file you modified was never intended to be run by itself it is run my tests/run.py. To run it we would run:./build/ALPHA_SE/m5.debug tests/run.py  long/00.gzip/alpha/tru64/simple-atomicNote that test.py isn't actually in there, and the run.py script figures out where test.py is based on the parameter to it.b) The SPEC binary you have is a tru64 binary not a linux binary (coff isn't used in linux), making the operating system parameter be linux is incorrect, you'll get the wrong system calls.c) cpu2000.py assumes a slightly different layout for the input files.
